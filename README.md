# 💰 SplitMaster Simple

**Мобильное приложение для удобного разделения расходов в группах**

---

## 📱 О проекте

SplitMaster — это современный инструмент для тех, кто хочет навести порядок в совместных расходах и при этом не терять удобства в общении. Проект прошел через фундаментальную трансформацию: я полностью переписал его с нуля, сменив технологический стек и архитектуру, чтобы обеспечить максимальную скорость и независимость.

## ✨ Основные возможности (Реализовано)

- 👥 **Групповые чаты в реальном времени** — мгновенный обмен сообщениями и медиафайлами.
- 📌 **Система Multi-Pin** — возможность закреплять несколько сообщений в чате для быстрого доступа.
- 🎨 **Кастомные аватарки** — уникальный движок для создания профилей из многослойных эмодзи.
- 💸 **Учёт расходов** — базовый функционал добавления трат и разделения их между участниками.
- 🇷🇺 **Полная локализация** — интерфейс и системная логика полностью на русском языке.
- ⚡ **Оптимистичный UI** — мгновенная реакция приложения на действия пользователя с фоновой синхронизацией.

## 🛠 Технологии (Текущий стек)

- **Flutter (Dart)** — кроссплатформенный фреймворк (вместо старого Kotlin), обеспечивающий плавный UI.
- **Self-hosted Supabase** — мой собственный бэкенд (вместо Firebase), дающий полную независимость и контроль над данными.
- **Realtime DB & Storage** — мгновенная синхронизация данных и хранение медиа на моих мощностях.
- **Material Design 3** — современный и чистый дизайн интерфейса.

## 🧭 Детальный Roadmap проекта

Моё видение развития **SplitMaster** на ближайшее время разделено на четыре ключевых этапа:

1. **Тотальная полировка**: Исправление абсолютно всех багов и доведение работы приложения до идеала. Пользовательский опыт должен быть безупречным.
2. **Завершение функционала чатов**: Полная реализация всех функций мессенджера (видео, голосовые, ответы, реакции, поиск).
3. **Функциональные группы**: Продвинутая система управления группами с ролями (админ/участник), логами активности и гибкой настройкой.
4. **Система финансового учета**: Интеграция выставления и подтверждения счетов. Это позволит контролировать все денежные потоки прямо в приложении.
